## My Home

Application de location de logements

- Les propriétaires de logements pouront pubier leurs biens
- Les utilisateurs pouront consulter la liste des logements et choisir ceux qu'ils préfèrent

Nous allons coder l'API de l'application


### Liste des dépendances

- composer require tymon/jwt-auth
    - Generer une clé secreet : php artisan jwt:secret
